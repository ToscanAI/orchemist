# Code of Conduct

This project adopts the Contributor Covenant, version 2.1, as the code
of conduct for all participation in the project (issues, PRs, reviews,
and any project-affiliated communication channels).

The authoritative text lives at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
Project participants are expected to follow that document.

## Reporting

Send reports to the maintainer contact listed in `MAINTAINERS.md`.

## Response timeline

Reports are acknowledged within 7 days of receipt and a resolution
decision is communicated within 30 days, in line with the enforcement
guidance of Contributor Covenant 2.1.
